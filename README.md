## Бібліотека `celebrate`

- Флоу валідації з `celebrate`
  - Маршрут > middleware валідації > контролер/middleware обробки помилок валідації
    - Якщо валідація проходить успішно, запит продовжує обробку до наступного middleware або контролера
    - Якщо помилка валідації виникає, вона передається до middleware обробки помилок
- Middleware `errors` для обробки помилок валідації
- Middleware валідації у маршрутах
- Організація файлів `src/validations`
- Схеми валідації з `Joi`
- Налаштування валідаціі з `Segments` для різних частин запиту (тіло, параметри маршруту, заголовки тощо)

## Валідація ідентифікатора MongoDB

- Хелпер `isValidObjectId`
- Валідатор `custom`
